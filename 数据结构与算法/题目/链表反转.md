## 二、单链表每隔K个反转

####题目：[leetCode链接](https://leetcode.com/problems/reverse-nodes-in-k-group/)

#### 实现

```java
思路：
先找到K个节点，记录head和tail,然后将这段节点按照单链表进行反转。
按照这个思路进行递归反转，最终将每段的尾和头连接，即完成了反转。

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        return reverseK(head,k);
    }
    
   // 按照k找到head、tail
    private ListNode reverseK(ListNode head,int k){
        if(null == head){
            return head;
        }
        ListNode tail = head;
        for(int i=1;i<k;i++){
            if(null == tail.next){
                return head;
            }
            tail = tail.next;
        }
        ListNode node = tail.next;
        reverseList(head,tail);
        head.next = reverseK(node,k);
        return tail;
    }
    
   // 单链表反转
    private void reverseList(ListNode head,ListNode tail){
        if(null == head || null == tail) {
            return;
        }
        ListNode pre = null;
        ListNode next = null;
        ListNode curr = head;
        while(curr != tail){
            next = curr.next;
            curr.next = pre;
            pre = curr;
            curr = next;
        }
        tail.next = pre;
    }
}
```
